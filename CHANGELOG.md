# Changelog

---

## v1.0.0

First public release.

**What's in it**
- Local Whisper transcription — tiny through large-v3
- Multi-file queue with drag-to-reorder
- Checkbox-select for targeted runs
- Run All / Run Selected
- Auto GPU detection (CUDA), graceful CPU fallback
- Translation mode (any language to English)
- Automatic language detection
- Export: TXT, SRT, VTT, JSON, TSV
- Merged transcript download with custom separators
- Search across all transcripts
- Timestamped segments view
- Editable transcript panel
- Copy and Copy Clean
- Real-time metadata: language, words, chars, segments, time taken
- Animated processing indicator
- Zero build step — Python + one HTML file
